# Multi-Site Company Network Design

A secure and scalable enterprise network built using **Cisco Packet Tracer**. The project simulates a company with two branches connected through a central router.

## Features

- VLANs
- Inter-VLAN Routing
- OSPF Routing
- DHCP
- DNS
- NAT
- Extended ACLs
- Port Security

## Network

- Two Branches
- Central Router
- Departmental VLANs (HR, IT, Sales, Finance)
- Central Web Server

## Technologies

- Cisco Packet Tracer
- Cisco IOS
- OSPF
- VLAN
- DHCP
- DNS
- NAT
- ACL
